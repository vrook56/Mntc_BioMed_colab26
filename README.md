MNTC BioMed Collaboration 2026
Moore Norman Technology Center — Student Project Gallery
A curated collection of student-built projects developed through the 2026 BioMed Collaboration at Moore Norman Technology Center (MNTC) in Norman, Oklahoma. This gallery showcases interactive, browser-based projects created by students working at the intersection of technology and biomedical concepts.
---
🔗 Live Site
View the gallery at:

📁 Project Structure
```
MNTC_BioMed_colab26/
├── index.html          # Main gallery landing page
├── student-name-1/
│   └── index.html      # Individual student project
├── student-name-2/
│   └── index.html
└── ...
```
Each student project lives in its own subfolder and is accessible directly via URL.
---
🧬 About the Collaboration
This gallery is the product of a cross-program collaboration between MNTC's Technology and BioMed departments. Students applied programming and data skills to biomedically-themed projects, demonstrating how technology serves the health sciences.
